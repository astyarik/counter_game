## Название

Игра со счётом

## Цель игры

Пройти все 30 волн, проходя волны с помощью бросков кубиков и грамотного использования перекрутов и атак.

---

# Игровой процесс

Каждая волна представляет собой отдельный счетчик с определённым количеством очков (HP).

В начале хода игрок получает случайную комбинацию из пяти кубиков со значениями от 1 до 5.

После победы начинается следующая волна.

---

# Основные характеристики

## HP

Количество очков текущей волны.

После каждой атаки очки уменьшаются на нанесённый урон.

Когда HP становится меньше или равно нулю, волна считается пройденной.

---

## Очки (сумма кубиков)

Очки вычисляются как сумма пяти выпавших чисел.

Например:

```
[4, 2, 5, 1, 3]

Сумма = 15
```

---

## Множитель

Базовый множитель рассчитывается по формуле:

```
волна + бонусный множитель
```

После этого могут добавляться дополнительные бонусы.

Если сумма кубиков делится на 2:

```
+2 к множителю
```

Если сумма делится на 3:

```
+3 к множителю
```

---

## Урон

Урон вычисляется по формуле:

```
(сумма кубиков + бонусные очки) × множитель
```

Именно это значение отнимается от HP волны.

---

# Перекруты

Перекрут полностью заменяет все пять кубиков на новые случайные значения.

Количество перекрутов ограничено.

После использования один перекрут расходуется.

---

# Атаки

Количество атак также ограничено.

Каждая атака:

* наносит урон;
* расходует одну попытку атаки;
* генерирует новый набор кубиков.

Если атаки заканчиваются до обнуления очков, игра считается проигранной.

---

# Волны

Всего в игре:

```
30 волн
```

После победы:

* увеличевается количество очков;
* восстанавливается количество атак;
* восстанавливается количество перекрутов.

---

# Улучшения

После каждой 4 волны игрок выбирает одно из двух улучшений.

### Вариант 1

```
+3 к сумме кубиков
```

Этот бонус добавляется ко всем последующим атакам.

### Вариант 2

```
+0.5 к множителю
```

Увеличивает итоговый множитель урона.

---

# Условия победы

Игрок побеждает, если успешно проходит все 30 волн.

После окончания игры выводится общее время прохождения.

---

# Условия поражения

Игрок проигрывает, если количество атак заканчивается раньше, чем очки станут равным нулю.

---

# Формулы

Множитель:

```
M = Волна + Бонусный множитель + Бонусы за делимость
```

Урон:

```
Урон = (Сумма + Бонус) * Множитель
```

---

# Особенности игры

* случайная генерация пяти кубиков;
* постепенное увеличение сложности;
* система постоянных улучшений;
* ограниченное количество атак;
* ограниченное количество перекрутов;
* стратегический выбор между риском и немедленной атакой.
